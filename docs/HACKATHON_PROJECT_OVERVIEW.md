# Tell us what your project does and why it's awesome

As AI keeps getting better, more industries will start adopting it. One of the key pioneers in the adoption of AI is the software engineering industry. "Vibe coding" tools have exploded onto the scene, each of the major players reaching millions of users.

But while all of these tools can build production-ready backends and integrate with APIs, not a single one of them is able to design appealing frontends.

To address this issue, I built Jasmine ([tryjasmine.dev](https://tryjasmine.dev)) — the world's first AI designer. Jasmine builds stellar frontend websites in a Vite+React or HTML/CSS/JS tech stack.

Unlike most vibe coding tools, Jasmine doesn't jump straight into building. It first analyzes the prompt and breaks it down into sections. It then identifies the target audience and goal of the website, and creates a full frontend design using its special design workflow.

Only after all of this is done, it starts coding.

On the technical side of things, Jasmine itself is built with Vite and React, uses multiple AI providers (Groq, Gemini, OpenAI) for generation, and runs live previews in E2B cloud sandboxes — no local build required. Users can chat to edit the output in real time, stream code as it's generated, and deploy to Netlify or download the project with one click. Auth and project storage run on Firebase.
