## 1. What is the project that you are currently working on or would like to pursue? Why? (Max: 500 characters)

## 2. What problem are you solving? (Max: 500 characters)

**Draft (≈320 chars):**

> AI code generators often produce incomplete, broken, or generic output. Users waste time fixing imports, missing files, and purple-gradient slop. Jasmine solves this by enforcing completeness (open-lovable style), streaming generations with instant preview, and running an automatic fix pass. The goal: describe what you want, get a working site.
